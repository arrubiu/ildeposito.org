
-- INTRODUCTION --

Linkit_views extends the Linkit module with the possibility to link to views pages

-- DEPENDENCIES --

Linkit_views has two obvious dependencies, Linkit and Views
* Linkit <http://drupal.org/project/linkit>
* Views <http://drupal.org/project/views>
